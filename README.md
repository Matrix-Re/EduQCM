## Project Overview

This platform enables teachers to create, manage, and assign multiple-choice questionnaires (QCM) to students.  
It also provides:

- student and teacher account management
- result tracking
- performance analytics

The **V2** introduces a modern architecture based on a **REST API + SPA**.
